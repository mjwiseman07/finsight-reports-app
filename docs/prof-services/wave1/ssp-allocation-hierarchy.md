# SSP Allocation Hierarchy

ASC 606-10-32-32/33/34 ordered approach (Q11=A):

1. Observable SSP (sold separately to similar customers in similar circumstances)
2. Adjusted market assessment (estimate at price market would pay)
3. Expected cost plus margin
4. Residual (only when SSP highly variable or uncertain AND other methods infeasible)

Top-down enforcement; the residual path is the last-resort gate. IFRS 15.78-79 mirrors this hierarchy.
