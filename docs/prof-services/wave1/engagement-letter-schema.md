# Engagement Letter Schema

Engagement-letter required-fields schema reference (reserved-channel preview):

- Parties + engagement scope
- Fee structure (fixed / retainer / contingent / value-based)
- Conflict-of-interest disclosure
- Confidentiality + privilege scope
- Termination + withdrawal terms
- Dispute resolution
- Signatures + dates

Per sub-segment specials:

- ABA Rule 1.5 (L)
- AICPA ET 1.310 (A)
- MCA standard (M)
- SOC 1/2 + HIPAA BAA + GDPR Art 28 attachments (I)
- PE-seal statement (E)
- work-for-hire + IP assignment (K)

Structural enforcement is reserved for PS-2 via the 9th audit channel `engagement-letter-audit`.
